 Sales Customer Intelligence Dashboard (Power BI)

Project Overview

This project is an interactive **Power BI dashboard** built to analyze sales performance, customer insights, and business trends.
It provides a clear and user-friendly interface for both overview and detailed analysis.

🔹 KPIs

* Total Sales
* Total Profit
* Profit Margin %
* Total Orders

🔹 Analysis

* Sales Trend with Forecast
* Sales by Region and Category
* Top N Products (Dynamic Parameter)
* Top Customers by Profit

 🔹 Advanced Features

* Conditional Formatting (Matrix Heatmap)
* Drillthrough for detailed insights
* Tooltip page with mini visual summaries
* Slicer panel with bookmarks (show/hide)
* Page navigation using buttons
* Row-Level Security (RLS)

 Dashboard Pages

1. Main Dashboard – Overview of key metrics
2. Product Analysis – Product performance insights
3. Customer Analysis – Customer behavior and profitability
4. Drillthrough Page – Detailed view for deeper analysis

 Tools Used

* Power BI
* DAX
* Excel
